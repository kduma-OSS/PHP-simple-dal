# Simple DAL - SQLite Adapter

SQLite database adapter for Simple DAL. Stores records as JSON in SQLite tables and attachments as BLOBs.

Part of the [Simple DAL](https://opensource.duma.sh/libraries/php/simple-dal) project.

## Installation

```bash
composer require kduma/simple-dal-db-adapter
```

See the [full documentation](https://opensource.duma.sh/libraries/php/simple-dal).
