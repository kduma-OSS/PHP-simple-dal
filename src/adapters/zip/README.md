# Simple DAL - ZIP Adapter

ZIP archive adapter for Simple DAL. Same layout as the directory adapter, but inside a ZIP file. Useful for export, import, and backups.

Part of the [Simple DAL](https://opensource.duma.sh/libraries/php/simple-dal) project.

## Installation

```bash
composer require kduma/simple-dal-zip-adapter
```

See the [full documentation](https://opensource.duma.sh/libraries/php/simple-dal).
