# Simple DAL - Encryption (Libsodium)

Libsodium encryption algorithm implementations for the Simple DAL encryption plugin. Provides SecretBox (symmetric) and SealedBox (asymmetric) algorithms.

Part of the [Simple DAL](https://opensource.duma.sh/libraries/php/simple-dal) project.

## Installation

```bash
composer require kduma/simple-dal-encryption-sodium
```

## Documentation

See the [full documentation](https://opensource.duma.sh/libraries/php/simple-dal).
