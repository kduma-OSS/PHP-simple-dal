# Simple DAL - Data Integrity (Libsodium)

Libsodium data integrity algorithm implementations for the Simple DAL data integrity plugin. Provides Blake2b (hashing) and Ed25519 (signing) algorithms.

Part of the [Simple DAL](https://opensource.duma.sh/libraries/php/simple-dal) project.

## Installation

```bash
composer require kduma/simple-dal-data-integrity-sodium
```

See the [full documentation](https://opensource.duma.sh/libraries/php/simple-dal).
