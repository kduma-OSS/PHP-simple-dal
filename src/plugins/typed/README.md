# Simple DAL - Typed Records Plugin

Adds strongly-typed PHP record classes to Simple DAL using `#[Field]` attributes and PHP 8.4 property hooks with automatic hydration and dehydration.

Part of the [Simple DAL](https://opensource.duma.sh/libraries/php/simple-dal) project.

## Installation

```bash
composer require kduma/simple-dal-typed
```

See the [full documentation](https://opensource.duma.sh/libraries/php/simple-dal).
