# Simple DAL - Encryption (phpseclib)

phpseclib3 encryption algorithm implementations for the Simple DAL encryption plugin. Provides RSA (asymmetric) and AES (symmetric) algorithms.

Part of the [Simple DAL](https://opensource.duma.sh/libraries/php/simple-dal) project.

## Installation

```bash
composer require kduma/simple-dal-encryption-phpseclib
```

See the [full documentation](https://opensource.duma.sh/libraries/php/simple-dal).
